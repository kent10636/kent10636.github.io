# Site Relaunch

After eight years of dormancy, my personal site is back — completely rebuilt from the ground up.

## What changed

The old site was a **Hexo 3.7.1** static export with the NexT Muse theme. It worked, but it was frozen in 2018: default titles, placeholder author names, and no source code in the repository.

The new stack:

- **Astro 6** for blazing-fast static generation
- **Tailwind CSS 4** for modern, responsive styling
- **MDX** for rich blog content
- **GitHub Actions** for automatic deployment

## Why Astro?

Personal sites are mostly content. Astro's islands architecture means the pages stay lightweight — JavaScript only loads where it's actually needed, like the theme toggle.

## What's next

I'll be adding more projects and blog posts over time. If you're reading this, the relaunch was a success.
